Intro to Programming

Lesson One: The basics of the web and HTML

Thinking like a programmer

  1. Procedural thinking: step 1, step 2, step 3
  2. Abstract thinking: 1+1=2, whether is is apples or oranges or houses you're counting.
  3. Technical empathy: Thinking like a computer.
  4. Systems thinking: Breaking down big ideas into pieces.

HTML: hypertext markup language

HTML elements are pieces of HTML documents. They are indicated with HTML tags.

Unlike humans, computers can't mentally fill in gaps if there are errors or information missing. A human has to be very careful there are no mistakes when giving directions to a computer.

HTML elements are either inline or block. Block elements form an invisible box around the content inside them.

Lesson Two: Creating a structured document with HTML

House analogy

HTML files - the layout of the house, like where the walls are

CSS files - the color of the carpeting, how the house is decorated

Javascript - interactive elements of the house, such at the TV remote and garage door opener

HTML language

HTML is a language with syntax and rules. It uses tags, which are like elements of a tree.

DOM: document object model

DOM is a standard convention for representing and interacting with elements in HTML.

Each HTML tag creates an element in the DOM that the browser uses to display the page.

CSS

CSS allows you to use syntax and rules to change how elements look on the page.

Lesson 3: Adding CSS style to HTML structure

Avoiding repetition

CSS lets programmers avoid repetition by assigning certain styles to all instances of a specific tag in the HTML.

This also allows programmers to avoid errors. For example, using CSS to make all text in h1 tags red, you won't accidentally make one h1 tag blue and make your web page inconsistent.

CSS: cascading style sheets

Cascading means that the most specific rule is applied to every element.

In real life, national laws > state laws > municipality laws > your household rules. If they were cascading, your household rules would also apply to national law.
